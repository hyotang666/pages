# PAGES 0.0.0
## What is this?
HTML Compiler for managing github pages.

## Usage

```shell
cd to/your/name.github.io ; Move to.
mkdir src                 ; Only first time.
vim src/blog.md           ; Writing.
pages                     ; Compiling.
git add .                 ; Adding.
git commit -m 'title'     ; Committing.
git push -u origin master ; Pushing.
```
